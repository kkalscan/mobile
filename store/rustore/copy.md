# KkalScan — тексты для RuStore Console

Скопируйте поля в [console.rustore.ru](https://console.rustore.ru) при загрузке версии.

## Название (≤50 символов, ASO)

```
KkalScan — калории по фото бесплатно
```

## Краткое описание

```
Сфоткай еду — узнай ккал и БЖУ. Поиск продуктов, дневник и графики. 3 скана в день бесплатно.
```

## Полное описание

```
Калории по фото бесплатно — сфотографируй тарелку и узнай ккал, белки, жиры, углеводы и клетчатку за секунды.

KkalScan — счётчик калорий и дневник питания: нейросеть распознаёт еду на фото, в том числе русские блюда. Можно добавить продукт через поиск, без ручного ввода калорий.

Что внутри:
• 3 бесплатных скана каждый день — завтрак, обед и ужин
• Дневник «Сегодня» с суммой ккал, БЖУ и клетчатки
• Поиск продуктов — быстро добавить блюдо в дневник
• Недельный дневник с графиками калорий, БЖУ и клетчатки
• Правка порций после скана — поправьте граммы за пару тапов
• Работа без регистрации — открыли и сразу сканируете
• Бонусные сканы за просмотр рекламы
• KkalScan Pro — безлимитные сканы и AI-разбор питания (199 ₽/мес)

Как пользоваться:
1. Нажмите «+» и выберите фото еды
2. Получите результат с ккал и БЖУ
3. Добавьте приём пищи в дневник — или найдите продукт через поиск

Приложение не заменяет консультацию врача или диетолога. Оценки калорий носят приблизительный характер и зависят от качества фото и состава блюда.

Ключевые темы: калории по фото, счётчик калорий, подсчёт калорий бесплатно, БЖУ, калькулятор калорий, распознавание еды, дневник питания, поиск продуктов, клетчатка, похудение, диета, бесплатно, нейросеть, русская еда.
```

## FAQ

| Вопрос | Ответ |
|--------|-------|
| Сколько сканов бесплатно? | 3 скана каждый день. Счётчик обновляется в полночь по вашему часовому поясу. |
| Нужна ли регистрация? | Нет. Приложение работает сразу после установки. |
| Можно добавить еду без фото? | Да — на экране «Сегодня» нажмите «Найти продукт» и выберите из каталога. |
| Насколько точны калории? | Это оценка нейросети. Для сложных блюд возможны отклонения. Не заменяет консультацию врача. |
| Как получить больше сканов? | Оформите KkalScan Pro (199 ₽/мес) или получите бонус за просмотр рекламы на экране лимита. |
| Куда отправляется фото? | На сервер KkalScan для анализа. Подробности — в политике конфиденциальности. |
