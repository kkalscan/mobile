# KkalScan — тексты для RuStore Console

Скопируйте поля в [console.rustore.ru](https://console.rustore.ru) при загрузке версии.

## Название (≤30 символов)

```
KkalScan — ккал по фото
```

## Краткое описание

```
Сфоткай еду — узнай калории и БЖУ. 3 бесплатных скана каждый день. Дневник без регистрации.
```

## Полное описание

```
Сфотографируй тарелку — получи калории и БЖУ за несколько секунд.

KkalScan помогает вести дневник питания без ручного поиска продуктов: сделайте фото еды, нейросеть оценит блюда и покажет ккал, белки, жиры и углеводы.

Что внутри:
• 3 бесплатных скана каждый день — хватит на завтрак, обед и ужин
• Дневник «Сегодня» с суммой калорий за день
• Работа без регистрации — открыли и сразу сканируете
• Бонусные сканы за просмотр рекламы
• KkalScan Pro — безлимитные сканы (подписка 199 ₽/мес)

Как пользоваться:
1. Нажмите «+» и выберите фото еды
2. Получите результат с ккал и БЖУ
3. Добавьте приём пищи в дневник

Приложение не заменяет консультацию врача или диетолога. Оценки калорий носят приблизительный характер и зависят от качества фото и состава блюда.

Ключевые темы: калории по фото, счётчик калорий, подсчёт калорий, БЖУ, распознавание еды, дневник питания, похудение, диета.
```

## FAQ

| Вопрос | Ответ |
|--------|-------|
| Сколько сканов бесплатно? | 3 скана каждый день. Счётчик обновляется в полночь по вашему часовому поясу. |
| Нужна ли регистрация? | Нет. Приложение работает сразу после установки. |
| Насколько точны калории? | Это оценка нейросети. Для сложных блюд возможны отклонения. Не заменяет консультацию врача. |
| Как получить больше сканов? | Оформите KkalScan Pro или получите +2 скана за просмотр рекламы на экране лимита. |
| Куда отправляется фото? | На сервер KkalScan для анализа. Подробности — в политике конфиденциальности. |
