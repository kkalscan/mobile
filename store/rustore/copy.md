# KkalScan — тексты для RuStore Console

Скопируйте поля в [console.rustore.ru](https://console.rustore.ru) при загрузке версии.

## Название (≤50 символов, ASO)

```
KkalScan — калории по фото бесплатно
```

## Краткое описание

```
Опишите еду текстом или сфоткайте — ккал и БЖУ без базы продуктов.
```

## Полное описание

```
Калории без рутины: не нужно искать блюдо в базе и листать каталог продуктов.

KkalScan — счётчик калорий и дневник питания. Опишите текстом, что съели — «тарелка борща», «хлеб 100 г», «кофе с молоком» — нейросеть оценит ккал, белки, жиры, углеводы и клетчатку. Или сфоткайте тарелку: распознавание русской еды за секунды.

Что внутри:
• Описать еду текстом — без выбора из каталога и ручного ввода калорий
• Скан по фото — 3 бесплатных распознавания каждый день
• Дневник «Сегодня» с суммой ккал, БЖУ и клетчатки
• Недельный дневник с графиками калорий, БЖУ и клетчатки
• Поиск продуктов — опционально, если знаете точное название
• Правка порций после скана или описания — поправьте граммы за пару тапов
• Работа без регистрации — открыли и сразу добавляете еду
• Бонусные сканы за просмотр рекламы
• KkalScan Pro — безлимитные сканы и AI-разбор питания (199 ₽/мес)

Как пользоваться:
1. «Сегодня» → карандаш → напишите, что съели — или «+» для фото
2. Получите результат с ккал и БЖУ
3. Добавьте приём пищи в дневник

Приложение не заменяет консультацию врача или диетолога. Оценки калорий носят приблизительный характер и зависят от качества фото или точности описания.

Ключевые темы: калории по фото, описать еду текстом, счётчик калорий, подсчёт калорий бесплатно, без базы продуктов, БЖУ, калькулятор калорий, распознавание еды, дневник питания, клетчатка, похудение, диета, бесплатно, нейросеть, русская еда.
```

## FAQ

| Вопрос | Ответ |
|--------|-------|
| Сколько сканов бесплатно? | 3 скана каждый день. Счётчик обновляется в полночь по вашему часовому поясу. |
| Нужна ли регистрация? | Нет. Приложение работает сразу после установки. |
| Можно добавить еду без фото? | Да — на «Сегодня» нажмите карандаш и опишите текстом («хлеб 100 г», «борщ»). Не нужно искать в базе продуктов. Или «Найти продукт», если знаете точное название. |
| Насколько точны калории? | Это оценка нейросети. Для сложных блюд возможны отклонения. Не заменяет консультацию врача. |
| Как получить больше сканов? | Оформите KkalScan Pro (199 ₽/мес) или получите бонус за просмотр рекламы на экране лимита. |
| Куда отправляется фото? | На сервер KkalScan для анализа. Подробности — в политике конфиденциальности. |
