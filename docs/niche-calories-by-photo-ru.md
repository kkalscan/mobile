# Ниша: подсчёт калорий по фото (Россия)

Документ зафиксирован на основе анализа Яндекс.Вордстат (мобильные запросы, янв 2024 — май 2026) и отзывов в RuStore / Google Play (июнь 2026).

**Целевой рынок:** Россия  
**Целевые платформы:** Android (RuStore, Google Play), iOS (App Store), веб (SEO)  
**Профиль разработчика:** соло, Android + iOS + веб

---

## 1. Суть ниши

Пользователь фотографирует еду → ИИ определяет блюда, граммы, калории и БЖУ → результат попадает в дневник питания.

Ключевое отличие от классических счётчиков (FatSecret, ручной ввод): **минимум действий**, без поиска продукта в базе и взвешивания.

### Формулировка продукта

> Бесплатный русский food scanner: сфоткал тарелку — получил ккал и БЖУ за 3 секунды.

## 5. Возможности для нового продукта

Маппинг болей → дифференциация:

| Боль пользователей | Возможность продукта |
|--------------------|----------------------|
| «Русскую кухню не знает» | Промпт + база русских блюд (борщ, гречка, плов, шаурма, столовая) |
| «Бесплатно одно блюдо» | 5–10 фото/день в free tier |
| «Деньги не вернули» | Прозрачная отмена, биллинг RuStore / СБП |
| «Каждый раз править» | UI правки граммов одним тапом, «исправить результат» |
| «Нет штрихкода» | Сканер штрихкода (у CalZen тоже нет — легко выделиться) |
| «Сразу платно» | Не просить отзыв и оплату в первые 5 минут |
| «Нет веба» | SEO-лендинг «калории по фото онлайн» (696 запросов) |
| YAZIO Pro недоступен | Фото в free tier, оплата в рублях |

### Чего не делать

- Не конкурировать с FatSecret по размеру базы продуктов.
- Не ставить только подписку без free tier (как Cal AI — 337 запросов в РФ).
- Не игнорировать RuStore.
- Не делать «для всего мира» на старте — русская еда = УТП.

## 8. MVP (ориентир 3–4 недели, соло)

### Неделя 1
- Камера → фото → Vision API (GPT-4o / YandexGPT Vision)
- Промпт под русские блюда
- Ответ: блюда + граммы + ккал + БЖУ

### Неделя 2
- Дневник питания (завтрак / обед / ужин)
- Правка граммов после распознавания
- Лимит 5 фото/день (freemium)

### Неделя 3
- iOS + Android
- RuStore + Google Play
- Лендинг с веб-версией «загрузи фото»

### Неделя 4
- Paywall (Точка API + rewarded ads)
- ASO по фразам из раздела 7

### Must-have по болям (v1)

- [ ] 5+ бесплатных сканов в день
- [ ] Правка граммов / ингредиентов после фото
- [ ] Промпт, заточенный под русскую еду
- [ ] Оплата в ₽ (RuStore)
- [ ] Без запроса отзыва до первого успешного скана
- [ ] Дневник + суточная норма ккал

### Nice-to-have (v2)

- [ ] Сканер штрихкода
- [ ] Счётчик воды
- [ ] Интеграция Health / Google Fit
- [ ] Статистика / графики
- [ ] Веб без установки приложения

## 9. Риски
