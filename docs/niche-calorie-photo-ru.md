# Ниша: подсчёт калорий по фото (Россия)

Документ зафиксирован по результатам анализа Яндекс.Вордстат и отзывов в RuStore / Google Play (июнь 2026).

**Целевой профиль разработчика:** соло, Android + iOS, веб-лендинг, фокус на российский рынок.

---

## 1. Суть ниши

Пользователь фотографирует еду → ИИ определяет блюда, граммы, ккал и БЖУ → результат попадает в дневник питания.

Ключевой запрос не «счётчик калорий» вообще, а именно **«калории по фото»** — отдельный быстрорастущий сегмент внутри категории health/fitness.

## 8. MVP (черновик, 3–4 недели, соло)

**Неделя 1:** камера → фото → Vision API → парсинг блюд/граммов/ккал/БЖУ (промпт под русскую еду)

**Неделя 2:** дневник питания, правка граммов, лимит 5 фото/день

**Неделя 3:** iOS + Android, RuStore + Google Play, лендинг с веб-загрузкой фото

**Неделя 4:** paywall (реклама; Точка — после HTTPS), ASO под ключевые фразы

### ASO-фразы

- калории по фото
- счётчик калорий
- подсчёт калорий бесплатно
- калории по фото бесплатно
- приложение для похудения
- бжу калькулятор

## 9. Риски

| Риск | Митигация |
|------|-----------|
| Точность ИИ ±15–30% | Показывать «примерно», правка граммов |
| Стоимость API на фото | Лимит free-сканов, кэш похожих блюд |
| CalZen уже лидер | Не копировать 1:1 — бить по болям из отзывов |
| Сезонность | Запуск к янв–мар или май–июл |

---

## 10. Отложенные направления (не в фокусе)

Из начала исследования обсуждались, но **не выбраны** как основная ниша:

- B2B-маршрутизатор для выездных мастеров
- Пешие / туристические маршруты
- VPN, MAX-мессенджер, общий AI-чат

Профиль разработчика: соло, быстрая разработка, Android + iOS + веб, Россия.
