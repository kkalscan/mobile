## 3. Утверждённый бренд

### Бренд: **КкалСкан** / **KkalScan**

| Слой | Значение |
|------|----------|
| **Бренд (RU)** | КкалСкан |
| **Бренд (EN/Latin)** | KkalScan |
| **Домен** | *позже* `kkalscan.ru` (не регистрируем на этапе MVP) |
| **API (сейчас)** | `http://91.207.75.72:8080/api/v1/` |
| **Оплата / privacy (сейчас)** | `http://91.207.75.72:8080/pay`, `/privacy` |
| **Android package** | `ru.kkalscan.app` (рекомендация) |
| **RuStore (название)** | `KkalScan — калории по фото бесплатно` |
| **Короткое описание** | Сфоткай еду — узнай ккал и БЖУ. 3 скана в день бесплатно. Дневник питания. |
| **Слоган** | *Сфоткай. Узнай ккал.* |
| **Telegram** | `@kkalscan` (проверить занятость) |
| **VK** | `vk.com/kkalscan` |

### Почему KkalScan

1. **Scan** — прямо про действие пользователя (фото еды), совпадает с продуктом v0.1.
2. **Ккал** — понятно на русском рынке.
3. **Нет ложного обещания** чата / диетолога (в отличие от КкалЧат).
4. **Отличие от CalZen** — другой корень (kkal + scan), узнаваемая иконка «камера + ккал».
5. **ASO:** короткий бренд + полная фраза в подзаголовке RuStore.

## 4. ASO (RuStore)

**Название (до 50 символов):**
```
KkalScan — калории по фото бесплатно
```

**Короткое описание:**
```
Опишите еду текстом или сфоткайте — AI посчитает ккал. Без базы продуктов. 3 скана/день.
```

**Ключи в полном описании:**
```
калории по фото, описать еду текстом, калории по фото бесплатно, счётчик калорий по фото,
подсчёт калорий, бжу, нейросеть, распознавание еды, без базы продуктов, дневник питания,
похудение, диета, сканер еды
```

**Не использовать:** CalZen, AIplate, взлом, 100% точность, ИИ-диетолог (нет в продукте).
